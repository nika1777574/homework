Python - პროგრამირების ენა

Python — პროგრამირების ენაა, რომელიც შეიქმნა გუიდო ვან როსუმის მიერ და პირველად გამოქვეყნდა 1991 წელს.

Python განკუთვნილია კოდის სიმარტივისთვის და წაკითხვისთვის, რის გამოც ის გამოიყენება როგორც დამწყებთათვის, ისე პროფესიონალების მიერ.

ენა მხარს უჭერს მრავალ პროგრამირების პარადიგმას, მათ შორის ობიექტურ-ორიენტირებულს, სტრუქტურულს და ფუნქციურს.

Python-ის სინტაქსი მიიჩნევა ძალიან მარტივად, ხოლო ინტერპრეტატორი საშუალებას აძლევს პროგრამისტს კოდი შეასრულოს ხაზობრივად, რაც ამარტივებს ტესტირებას და ექსპერიმენტირებას.
