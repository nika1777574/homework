HTML

HTML (HyperText Markup Language) წარმოადგენს ჰიპერტექსტური მარკირების ენას, რომელიც ძირითადად გამოიყენება ინტერნეტის ქსელის დოკუმენტების შესაქმნელად. HTML სათავეს იღებს გასული საუკუნის 90-იანი წლებიდან, როგორც ვებგვერდების შესაქმნელი პრიმიტიული ენა, ხოლო დღეისთვის ინტერნეტი წარმოუდგენელია ამ ენის გარეშე. საიტების აბსოლუტური უმრავლესობა ასე თუ ისე იყენებს HTML-ს.
